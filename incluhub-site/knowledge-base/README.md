# Knowledge Base Folder

Use `incluhub_website_kb.md` as the single source of truth for the Dedicated Manager chat.

## Workflow
1. Open `knowledge-base/incluhub_website_kb.md`
2. Add/update facts about products, roles, policies, contact flow, FAQs
3. Save the file
4. If dev server is running, refresh the page to use latest content

## Notes
- Keep content factual and sectioned with clear headings.
- Do not add sensitive credentials or private secrets.
- The chatbot uses this file for retrieval-based context grounding.
